## 4. Data Handling Rules

- Upload only into the assigned project UAT environment.
- Do not reuse client information in demos, marketing, samples or model training without written authorization.
- Use role-based and time-limited access.
- Keep access logs and upload/download logs.
- Remove access after UAT closeout.
- Confirm backup, restore and secure deletion requirements before production use.
